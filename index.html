Publer ID Finder

貼入 Publer API Key，即時列出 Workspaces、Accounts、Instagram Publer Account ID，並產生 GitHub Secret / Variable commands。API Key 預設留空，不會寫入此 HTML 檔案。

1) 輸入 Publer API Key

Browser 直接 call API 有機會被 CORS 擋住。若失敗，下面會自動產生 Terminal curl command，你照貼去 Terminal 跑就得。
等待輸入 API Key。

2) 最終你要放入 GitHub 嘅三個名稱

官方/API 對應：
GitHub NamePubler / API 名稱
PUBLER_API_KEYAPI key / Bearer API token
PUBLER_WORKSPACE_IDWorkspace ID / Publer-Workspace-Id
PUBLER_ACCOUNT_IDSSocial account ID / account id
重點：PUBLER_ACCOUNT_IDS 要用 Publer 回傳嘅 id，唔係 Instagram username，亦唔係 social_id。
